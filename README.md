A multi-agent wellness assistant that provides guidance on hydration, nutrition, and exercise. Combines lightweight rule-based agents with an optional LLM fallback for general wellness questions.  

**Features:**  
- Hydration, nutrition, and exercise advice  
- Session-based memory for context-aware responses  
- Optional LLM integration for fallback answers  

**Setup:**  
1. Install dependencies: `pip install -r requirements.txt`  
2. Set your GENIE_API_KEY in the environment or directly in the code.  
3. Run the agent: `python app.py`  

Interact with the agent via the console to get personalized wellness tips.
